# Code of Conduct

## Our Commitment

Paxeer Network is committed to fostering an open, welcoming, and inclusive environment for all contributors to the Orbex Exchange project. We expect all participants to uphold these standards in all project spaces, including GitHub repositories, issue trackers, discussion forums, and any other communication channels.

---

## Standards of Behavior

### Expected Conduct

- **Professionalism** - Communicate respectfully and constructively in all interactions
- **Inclusivity** - Welcome contributors of all backgrounds, experience levels, and perspectives
- **Collaboration** - Work together toward shared goals and accept constructive feedback gracefully
- **Accountability** - Take responsibility for mistakes and focus on resolution rather than blame
- **Respect** - Value differing viewpoints and experiences within the community

### Unacceptable Conduct

- Harassment, discrimination, or intimidation of any kind
- Personal attacks, insults, or derogatory comments
- Publishing private information without explicit consent
- Trolling, inflammatory remarks, or deliberately disruptive behavior
- Sexual language, imagery, or unwelcome advances
- Any conduct that would be considered inappropriate in a professional setting

---

## Scope

This Code of Conduct applies to all project spaces managed by Paxeer Network, including:

- GitHub repositories and issue trackers
- Pull request discussions and code reviews
- Project documentation and wikis
- Official communication channels (Discord, Telegram, etc.)
- Public representations of the project at events or online

This Code also applies when an individual is officially representing the project in public spaces.

---

## Enforcement

### Reporting

Instances of unacceptable behavior may be reported to the Paxeer Network team at:

**Email:** conduct@paxeer.network

All reports will be reviewed promptly and handled with discretion. The reporter's identity will be kept confidential unless disclosure is required by law.

### Response Process

1. **Acknowledgment** - Reports will be acknowledged within 48 hours
2. **Investigation** - The team will investigate the reported behavior
3. **Determination** - A decision will be made regarding appropriate action
4. **Communication** - The reporter will be informed of the outcome

### Consequences

Violations of this Code of Conduct may result in:

- **Warning** - A private written warning with clarification of the violation
- **Temporary Suspension** - Temporary restriction from project spaces
- **Permanent Ban** - Permanent removal from all project spaces

The severity of the response will be proportional to the nature and impact of the violation.
