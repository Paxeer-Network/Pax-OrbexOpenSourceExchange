## [1.0.0] - 2025-12-12

### Added

- **Initial Open Source Release** - Complete cryptocurrency exchange platform released to the community
- **Spot Trading** - Full order book trading with limit, market, and stop orders
- **P2P Trading** - Peer-to-peer marketplace with escrow and dispute resolution
- **Futures Trading** - Leveraged perpetual and dated futures contracts
- **Binary Options** - Time-based option contracts with configurable durations
- **ICO Launchpad** - Multi-phase token sale platform with allocation tiers
- **NFT Marketplace** - Create, buy, sell, and auction digital assets
- **Staking Pools** - Flexible and locked staking with reward distribution
- **AI Investment** - Automated portfolio management and trading strategies
- **Forex Investment** - Signal-based forex trading accounts
- **Multi-Network Wallets** - Support for EVM, Solana, Tron, TON, and Bitcoin
- **KYC/AML System** - Identity verification with customizable templates
- **Admin Dashboard** - 755+ management endpoints for platform control
- **MLM/Referral System** - Binary and unilevel referral reward structures
- **E-commerce Module** - Physical and digital goods marketplace
- **Notification System** - Email, SMS, and push notifications
- **Multi-language Support** - i18n internationalization framework
- **Theme System** - Customizable UI themes and components
- **Go Matching Engine** - High-performance order matching (optional)

### Technical Stack

- Next.js 15 with React 18 frontend
- Node.js/TypeScript backend with 1,172+ API files
- 110+ Sequelize ORM models
- MySQL primary database
- ScyllaDB for high-throughput trading data
- Redis for caching and real-time pub/sub
- uWebSockets.js for WebSocket streaming
- Go matching engine for high-frequency environments

## Release Notes

### Versioning Policy

- **Major versions** (X.0.0) - Breaking changes or significant architectural updates
- **Minor versions** (0.X.0) - New features and enhancements
- **Patch versions** (0.0.X) - Bug fixes and security patches

### Support Policy

Each major version receives security updates for 12 months following the release of the subsequent major version.
